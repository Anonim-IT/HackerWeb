## 1. Что такое Matrix и зачем это нужно

**Matrix** — это открытый, децентрализованный протокол для безопасного общения. Это как Telegram или WhatsApp, но:

- ✅ **Вы владеете своими данными** (сервер принадлежит вашему другу/сообществу)
- ✅ **Сквозное шифрование** — даже администратор сервера не может прочитать ваши сообщения
- ✅ **Федерация** — вы можете общаться с людьми на других Matrix-серверах (matrix.org, etc.)
- ✅ **Бесплатно** — нет рекламы, не продают ваши данные

**Ваш сервер:** `hwpro.online`  
**Администратор:** ваш знакомый, который его настроил

## 2. Регистрация аккаунта

Регистрация происходит **через клиент** (приложение). Не нужно ничего вводить на сайте.

### Для регистрации вам понадобится:
- **Адрес сервера:** `hwpro.online`
- **Придумать логин** (только латиница, цифры, символы `._-`)
- **Придумать надёжный пароль**

### ВАЖНО: как регистрироваться в зависимости от клиента

| Клиент | Где ввести адрес сервера |
|--------|-------------------------|
| **Element Web** | На странице входа → "Изменить сервер" → ввести `hwpro.online` |
| **Element Desktop** | На странице входа → "Изменить сервер" → ввести `hwpro.online` |
| **Element iOS/Android** | Нажать "Изменить сервер" при входе → ввести `hwpro.online` |
| **Другие клиенты** | Искать поле "Homeserver URL" или "Сервер" |

## 3. Установка и настройка клиента Element

**Element** — официальный и самый популярный клиент для Matrix.

### 3.1 Скачивание

| Платформа | Ссылка |
|-----------|--------|
| **Web** | https://app.element.io (но лучше использовать ваш сервер!) |
| **Windows / macOS / Linux** | https://element.io/download |
| **iOS** | App Store → ищи "Element" |
| **Android** | Google Play / F-Droid → ищи "Element" |

### 3.2 Первый вход (пошагово)

1. **Запустите Element**

2. **Нажмите "Изменить сервер"** (Change server) — это важно!

3. **В поле "Домашний сервер" (Homeserver URL) введите:**
   ```
   https://hwpro.online
   ```

4. **Нажмите "Зарегистрироваться" (Register)**

5. **Заполните форму:**
   - **Логин (Username):** придумайте, например: `anna`, `pavel_2026`, `friend42`
   - **Пароль (Password):** надёжный пароль (используйте менеджер паролей!)
   - **Подтверждение пароля**
   - **Email (опционально):** можно указать для восстановления

6. **Нажмите "Зарегистрироваться"**

7. **Готово!** Вы внутри.

## 4. Первые шаги после входа

### 4.1 Интерфейс

- **Левая панель:** список чатов и комнат
- **Центральная область:** сообщения
- **Правая панель:** информация о комнате/пользователе
- **Аватар в левом верхнем углу:** настройки и выход

### 4.2 Найти и написать администратору

В поле поиска (вверху слева) введите:
```
@admin:hwpro.online
```
(замените `admin` на логин администратора, который вам дали)

### 4.3 Создать свою первую комнату

1. Нажмите **"+"** рядом с "Комнаты"
2. Выберите **"Создать новую комнату"**
3. Введите название комнаты
4. Выберите приватность:
   - **Приватная** — только по приглашениям
   - **Публичная** — можно найти в поиске

## 5. Настройка безопасности (ВАЖНО!)

**Это самый важный раздел!** Без этих настроек вашу переписку могут прочитать.

### 5.1 Включение шифрования

1. Нажмите на **аватар** → **Все настройки**
2. Перейдите в **Безопасность и конфиденциальность**
3. Включите:
   - ✅ **Шифровать нешифруемые сообщения автоматически**
   - ✅ **Запрашивать подтверждение при входе с нового устройства**

### 5.2 Создание и сохранение ключа восстановления (КРИТИЧНО!)

**Ключ восстановления — это единственный способ расшифровать вашу историю, если вы потеряете доступ к устройствам.**

1. В настройках → **Безопасность** найдите раздел **"Ключ восстановления"**
2. Нажмите **"Создать ключ"**
3. **Скопируйте ключ в надёжное место!**

**Где НЕ хранить ключ:**
- ❌ В заметках телефона
- ❌ На том же устройстве, где установлен Element
- ❌ В облаке без шифрования

**Где хранить:**
- ✅ **Распечатайте и положите в сейф**
- ✅ **Напишите от руки на бумаге**
- ✅ **Используйте офлайн-менеджер паролей** (KeePassXC на флешке)

**Пример записи:**
```
Matrix Recovery Key
Пользователь: anna
Сервер: hwpro.online
Ключ: ESCX-ABCD-1234-EFGH-5678-IJKL-9012-MNOP
Дата: 29.03.2026
```

### 5.3 Верификация устройств

Когда вы войдёте с другого устройства (телефон, другой компьютер), Element попросит подтвердить новое устройство.

**Как верифицировать:**
1. На старом устройстве появится запрос: "Кто-то вошёл с нового устройства"
2. Нажмите **"Проверить"**
3. Выберите **"Сравнить эмодзи"**
4. Убедитесь, что 4 эмодзи совпадают на обоих устройствах
5. Нажмите **"Подтвердить"**

### 5.4 Управление сессиями

Периодически проверяйте список устройств:
- Настройки → **Безопасность** → **Управление сессиями**
- Удалите устройства, которыми больше не пользуетесь

## 6. Как общаться

### 6.1 Основные команды и жесты

| Действие | Как сделать |
|----------|-------------|
| **Начать чат** | Нажать "+" → "Начать чат" → ввести логин пользователя |
| **Создать комнату** | "+" → "Создать новую комнату" |
| **Пригласить в комнату** | Нажать на название комнаты → "Пригласить" → ввести логин |
| **Форматирование** | Используйте Markdown: `*курсив*`, `**жирный**`, `[ссылка](url)` |
| **Вставка кода** | \`\`\` код \`\`\` |
| **Упоминание** | @username:сервер (например, @anna:hwpro.online) |
| **Редактировать сообщение** | Нажать на сообщение → значок карандаша |
| **Удалить сообщение** | Нажать на сообщение → корзина |
| **Ответить на сообщение** | Нажать на сообщение → стрелка влево |

### 6.2 Как найти человека на другом сервере

Если ваш друг на другом сервере (например, matrix.org):
```
@друг:matrix.org
```
Введите это в поиск, и начните чат как обычно.

### 6.3 Комнаты с другими серверами

Чтобы присоединиться к публичной комнате на другом сервере:
```
#название_комнаты:matrix.org
```

## 7. Решение частых проблем

### 7.1 "Не могу зарегистрироваться — адрес сервера не принимает"

**Решение:**
- Убедитесь, что ввели `https://hwpro.online` (с https://)
- Попробуйте другой клиент (например, веб-версию)
- Свяжитесь с администратором — возможно, регистрация временно закрыта

### 7.2 "Забыл пароль"

Если вы настроили email при регистрации:
1. На странице входа нажмите **"Забыли пароль?"**
2. Введите email
3. Следуйте инструкциям в письме

Если email не указан — только администратор сервера может сбросить пароль (но это удалит зашифрованную историю).

### 7.3 "Потерял все устройства и ключ восстановления"

**К сожалению, расшифровать историю будет невозможно.** Это особенность сквозного шифрования.  
Вы сможете:
- Создать новый аккаунт
- Попросить администратора удалить старый

### 7.4 "Не могу прочитать старые сообщения на новом устройстве"

**Причина:** устройство не верифицировано или нет ключа восстановления.

**Решение:**
1. Если есть старое устройство — верифицируйте новое через него
2. Если нет старого устройства — используйте **ключ восстановления** (пункт 5.2)

### 7.5 "Сообщения приходят с серой иконкой замка"

Это значит, что сообщение **не зашифровано**.  
- Включите шифрование в настройках (пункт 5.1)
- Создайте новую комнату с включённым шифрованием

## 📱 Советы для разных устройств

### Для телефона (iOS/Android)
- Включите уведомления в настройках системы
- Используйте PIN-код или биометрию для блокировки приложения
- Регулярно проверяйте список сессий

### Для компьютера (Desktop)
- Не оставляйте Element включённым на публичных компьютерах
- Используйте мастер-пароль, если встроенный
- Выходите из приложения, если уходите от компьютера

### Для браузера (Web)
- **Не рекомендуется для постоянного использования** — закрытие браузера может привести к потере ключей
- Используйте только если нет возможности установить приложение
