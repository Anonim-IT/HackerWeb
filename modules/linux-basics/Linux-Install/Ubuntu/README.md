# Установка Ubuntu 25.10 на стационарный ПК

Данное руководство поможет вам установить операционную систему Ubuntu 25.10 на стационарный компьютер. Процесс описан пошагово, от создания загрузочной флешки до первого запуска системы.

## 📋 Системные требования

Для комфортной работы рекомендуется следующая конфигурация:

| Компонент | Минимальные требования | Рекомендованные требования |
|-----------|------------------------|----------------------------|
| **Процессор** | 2 ГГц двухъядерный | 4-ядерный от 3 ГГц |
| **Оперативная память** | 4 ГБ | 8 ГБ и более |
| **Место на диске** | 25 ГБ свободного места | 100 ГБ и более (SSD) |
| **Видеокарта** | С поддержкой VGA/HDMI | Дискретная NVIDIA/AMD для игр или рендеринга |
| **Интернет** | Не обязателен | Рекомендуется для загрузки обновлений |

## 🛠 Необходимые материалы

1.  **ISO-образ** Ubuntu 25.10 (скачать с [официального сайта](https://ubuntu.com/download))
2.  **USB-накопитель** (флешка) объемом от **8 ГБ**
3.  **Доступ в BIOS/UEFI** материнской платы

## 📝 Инструкция по установке

### Шаг 1: Создание загрузочной флешки

1.  Вставьте USB-накопитель в компьютер.
2.  Скачайте и установите программу для записи образов (рекомендуем **Rufus** для Windows или **Balena Etcher** для Windows/macOS/Linux).
3.  Запустите программу, выберите скачанный ISO-файл Ubuntu и ваш USB-накопитель.
4.  Для **Rufus**: убедитесь, что схема раздела установлена **GPT** (для UEFI) или **MBR** (для Legacy BIOS), нажмите "Старт".
5.  Дождитесь окончания записи.

### Шаг 2: Подготовка ПК (Настройка BIOS/UEFI)

1.  Перезагрузите компьютер.
2.  Во время старта нажмите клавишу входа в BIOS: обычно это **Del**, **F2**, **F10** или **F12** (зависит от производителя материнской платы).
3.  В настройках найдите раздел **Boot** (Загрузка).
4.  Установите приоритет загрузки на **USB Flash Drive**.
5.  **Важно:** Если вы устанавливаете современную версию Ubuntu, отключите **Secure Boot** (защищенная загрузка) или оставьте включенным (Ubuntu его поддерживает, но с проприетарными драйверами могут возникнуть проблемы).
6.  Сохраните изменения (F10) и выйдите.

### Шаг 3: Запуск установки

1.  После перезагрузки появится меню **GRUB**. Выберите *"Try or Install Ubuntu"* (Попробовать или установить Ubuntu).
2.  Выберите язык установки (рекомендуется *Русский* или *English*).
3.  Настройте раскладку клавиатуры.
4.  Выберите тип установки:
    *   **Интерактивная установка** (рекомендуется новичкам) — позволит настроить среду (выбор программ, кодеков, драйверов).
    *   **Автоматическая установка** — для продвинутых пользователей.

### Шаг 4: Выбор диска и разметка

В зависимости от вашей цели:

**Вариант А: Установка на чистый ПК (единственная система)**
*   Выберите пункт *"Стереть диск и установить Ubuntu"*.
*   Система автоматически разметит диск, создав необходимые разделы (EFI, `/`, `swap` (если нужно)).

**Вариант Б: Установка рядом с Windows (Dual Boot)**
*   Выберите *"Установить Ubuntu рядом с Windows Boot Manager"*.
*   Настройте размер для Ubuntu, потянув ползунок между разделами.

**Вариант В: Ручная разметка (опытные пользователи)**
*   Выберите *"Другой вариант"*.
*   Создайте таблицу разделов GPT (если диск пуст).
*   Минимальная структура:
    *   `/boot/efi` (если UEFI) — размер 512 МБ, тип FAT32.
    *   `/` (root) — размер от 30 ГБ, тип Ext4.
    *   `swap` — размер равен объему ОЗУ (для гибернации) или 4-8 ГБ.

### Шаг 5: Завершение настройки

1.  Укажите **часовой пояс**.
2.  Создайте **пользователя**:
    *   Имя компьютера (обычно латиницей).
    *   Имя пользователя (логин).
    *   Надежный пароль.
    *   (Опционально) Включите шифрование домашней папки.
3.  Нажмите *"Установить"*.
4.  Дождитесь окончания копирования файлов. Когда установка завершится, нажмите *"Перезагрузить сейчас"*.

### Шаг 6: Первый запуск и настройка

1.  После перезагрузки извлеките USB-флешку, когда увидите соответствующее сообщение.
2.  Войдите в систему под созданным пользователем.
3.  **Обновление системы:**
    Откройте терминал (`Ctrl + Alt + T`) и выполните:
    ```bash
    sudo apt update && sudo apt upgrade -y
    ```
4.  **Установка драйверов (особенно важно для NVIDIA):**
    Откройте *"Программы и обновления"* -> вкладка *"Дополнительные драйверы"*. Выберите проприетарный драйвер (tested/proprietary) вместо открытого `nouveau`.

## 🚀 Рекомендации после установки

1.  **Установка кодеков:**
    ```bash
    sudo apt install ubuntu-restricted-extras
    ```
2.  **Настройка брандмауэра (UFW):**
    ```bash
    sudo ufw enable
    ```
3.  **Установка любимого ПО:**
    *   Для установки программ используйте **App Center** (Магазин приложений) или терминал:
    ```bash
    sudo apt install vlc gimp git curl
    ```

## ⚠️ Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| **Черный экран после установки (NVIDIA)** | При загрузке в меню GRUB нажмите `e`, найдите строку с `quiet splash`, добавьте `nomodeset`, нажмите `F10`. После загрузки установите драйверы NVIDIA. |
| **Не видит SSD (NVMe) или другой диск** | Убедитесь, что в BIOS режим SATA установлен в **AHCI**, а не **Intel RST (RAID)**. |
| **Нет звука через HDMI** | Установите драйверы звука PulseAudio или проверьте настройки вывода звука в настройках системы. |
| **Wi-Fi не работает** | Подключитесь по проводу и выполните обновление системы. Часто драйверы подтягиваются автоматически. |
